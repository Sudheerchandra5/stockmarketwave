When to use this calculator

Use this STP calculator when planning systematic transfers from one mutual fund to another, such as moving from a liquid fund into an equity fund over time.

How to interpret the result

The calculator estimates total transferred amount, remaining source balance, and target value using expected return assumptions for both funds.

Common mistakes to avoid

Check fund house rules, exit loads, taxes, and minimum transfer amounts before starting an STP. Actual returns can differ from planned assumptions.

FAQ

Frequently asked questions

What is the STP Calculator used for?

The STP Calculator is a free mutual funds calculator that works out total transferred from source fund amount, monthly transfer, source return (%), target return (%), and years. Enter your own figures to get an instant result with the formula shown on the page.

What formula does the STP Calculator use?

It calculates Total transferred = Monthly transfer x Months and Target value estimate = Transferred amount x Growth factor. The tool reports total transferred, estimated source balance, estimated target value, and monthly transfer.

Where do I get the numbers for the STP Calculator?

Take source fund amount, monthly transfer, source return (%), target return (%), and years from your fund statement or scheme factsheet. The STP Calculator never fetches live market data, so the output depends only on the values you type in.

Are the STP Calculator results financial advice?

No. The STP Calculator is an educational tool. Treat total transferred as an estimate, re-check your inputs, and confirm important investment decisions with a qualified professional.

Can I use the STP Calculator with any currency or market?

Yes. Keep source fund amount, monthly transfer, source return (%), target return (%), and years in the same currency and consistent units, and the STP Calculator applies the same calculation to any market.
