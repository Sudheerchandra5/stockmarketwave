FAQ

Frequently asked questions

What is the Real Return Calculator used for?

The Real Return Calculator is a free mutual funds calculator that works out real annual return from nominal return (%), inflation (%), and years. Enter your own figures to get an instant result with the formula shown on the page.

What formula does the Real Return Calculator use?

It calculates Real return = (1 + Nominal return) / (1 + Inflation) - 1. The tool reports real annual return, real growth factor, nominal return, and inflation.

Where do I get the numbers for the Real Return Calculator?

Take nominal return (%), inflation (%), and years from your fund statement or scheme factsheet. The Real Return Calculator never fetches live market data, so the output depends only on the values you type in.

Are the Real Return Calculator results financial advice?

No. The Real Return Calculator is an educational tool. Treat real annual return as an estimate, re-check your inputs, and confirm important investment decisions with a qualified professional.

Can I use the Real Return Calculator with any currency or market?

Yes. Keep nominal return (%), inflation (%), and years in the same currency and consistent units, and the Real Return Calculator applies the same calculation to any market.
